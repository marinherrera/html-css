Parágrafos e quebras de linhas

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no mieo do par de tags "<p e /p>".

Se precisar "quebrar" o texto (ir para próxima linha, por exemplo) em algum lugar especifico como esse, é só usar a tag "br".

Para fazer os símbulos das tags (setas para os lados) basta escrever:
& l t ; = <
& g t ; = >
